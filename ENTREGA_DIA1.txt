1️⃣  POST /api/v1/auth/login
    ├─ Input: { email, password }
    ├─ Output: { access_token, token_type, expires_in }
    ├─ Status: 200 OK o 401 Unauthorized
    └─ Security: bcrypt + HS256 JWT (30 min)

2️⃣  POST /api/v1/auth/register
    ├─ Input: { email, password, nombre }
    ├─ Output: { id, email, nombre, is_active, roles }
    ├─ Status: 201 Created o 422 Unprocessable Entity
    └─ Auto-assign: Rol CLIENT

3️⃣  GET /api/v1/auth/me
    ├─ Auth: Bearer token (HTTP Authorization header)
    ├─ Output: { id, email, nombre, is_active, roles }
    ├─ Status: 200 OK o 401 Unauthorized
    └─ Protected: require_current_user

   • Expiration: 30 minutos
   • Payload: { sub: user_id, exp, iat }

   • Algorithm: bcrypt
   • Cost factor: 12 (SECURE - mínimo recomendado)
   • Timing-safe: comparison protegida

✅ Autenticación
   • HTTP Bearer token scheme
   • Validación en cada request
   • Error 401 automático

✅ CORS Middleware
   • Configurable desde .env
   • Credentials: true
   • Methods: ["*"]
   • Headers: ["*"]

✅ Validación Pydantic
   • Email: EmailStr (formato + único en BD)
   • Password: min 8, max 255
   • Nombre: min 2, max 100
   • Auto-documentation en Swagger
